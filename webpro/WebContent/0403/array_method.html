옥혜원 바보

indexOf/lastIndexOf

arr = ["사과", "바나나", "복숭아", "참외", "사과", "딸기", "포도", "사과", "복숭아"];
indexOf(item, start) : item의 위치를 검색
start : 시작위치, 생략시 0번째부터 시작
lastIndexOf(item) : item의 위치를 뒤에서부터 검색



push/pop

push : 배열의 끝에 추가
pop : 배열의 끝에서 제거


reverse/slice

reverse : 배열의 순서를 뒤집기
slice(start, end) : start, end 사이의 값을 추출(end번째는 포함되지 않음), 새로운 배열 만들기
splice : 배열 수정
- splice(idx, 삭제 개수, 새로운 데이터) : 기존 데이터를 삭제하지 않고 새로운 데이터를 삽입
- 기존 데이터 삭제하지 않고 새로운 데이터를 삽입
- 기존 데이터 삭제 후 새로운 데이터를 삽입


splice

splice : 배열 수정
- splice(idx, 0, 새로운 데이터) : 기존 데이터를 삭제하지 않고 새로운 데이터를 삽입
- splice(idx, 삭제 개수, 새로운 데이터) : 기존 데이터를 삭제 후 새로운 데이터를 삽입
- splice(idx, 삭제 개수) : 기존 데이터를 삭제
