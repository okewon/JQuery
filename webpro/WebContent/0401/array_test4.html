script 안에서 함수 밖에서 맨처음 선언되는 변수는
		var키워드가 있거나 없거나 전역변수이다.
		모든 함수 내에서 사용가능하다.

배열 1

리터럴(상수값)에 의한 배열 생성
var arr = ["사과", "바나나", "참외", "딸기", "수박"];

배열 2

생성자에 의한 배열 생성
var arr = new Array("사과", 123, 123.456, "딸기", "수박");

배열 3

생성자에 의한 배열 생성
var arr = new Array();
arr[0] = "사과";
arr[1] = 100;
arr[2] = "딸기";
