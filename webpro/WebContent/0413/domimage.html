이미지 변경하기

- 버튼을 클릭할때마다 이미지를 변경한다.
- 이미지는 배열로 구성하고 랜덤으로 선택한다.

이미지 변경하기

- 버튼을 클릭하면 이미지를 변경한다.
- 이미지는 배열로 구성하고 랜덤으로 선택한다.
- 1초에 한번씩 이미지를 변경

이미지 변경하기

- 두개의 이미지를 번갈아가면서 변경한다.
- 이미지를 클릭하면 다른 이미지로 바꾼다.
- match() : 특정 문자가 포함되어 있느냐
있다 : 문자가 리턴
없다 : null 리턴

- 사용자는 가위바위보를 입력
