위치나 글자 찾기

str = "우리 나라 좋은 나라 대한 민국";
indexOf(item, start) : item의 위치를 반환
charAt(idx) : idx 번째의 글자를 반환
charCodeAt(idx) : idx번째의 유니코드(ASCII) 반환

글자 추출 찾기

str = "우리 나라 좋은 나라 대한 민국";
substr(start, length) : start에서 length(갯수)만큼 추출
substring(start, end) : start에서 end사이의 문자를 추출
slice(start, end) : start에서 end사이의 문자를 자르기
split(구분자, 갯수) : 구분자로 문자열을 구분, 배열을 리턴
